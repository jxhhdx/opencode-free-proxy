# 🗂 功能清单 — OpenCode Free Proxy

> Rust / Tauri 桌面端应用 · 双击即用，无需配环境

---

## ✅ 已实现

### 1. 🌐 代理服务器
- [x] **OpenAI 格式** — `POST /v1/chat/completions`
- [x] **Anthropic 格式** — `POST /v1/messages`
- [x] **模型列表** — `GET /v1/models`
- [x] **健康检查** — `GET /health`
- [x] **流式输出 (SSE)** — 两种格式均支持 `stream: true`
- [x] **CORS** — 任意跨域请求
- [x] **请求/响应转换** — Anthropic ↔ OpenAI 格式互转（含 tool calls）

### 2. 🔑 API Key 管理
- [x] **自动生成** — 首次运行自动创建 admin / user-default 两个 key
- [x] **持久化存储** — 保存到 `~/.config/opencode-free-proxy/api-keys.json`
- [x] **双认证方式** — 支持 `Authorization: Bearer` 和 `x-api-key` 请求头
- [x] **仪表盘展示** — 打开应用即可看到所有 key
- [x] **一键复制** — 点击 Key 旁边的 📋 按钮复制到剪贴板

### 3. 📋 桌面仪表盘
- [x] **服务器状态指示器** — 绿色/红色/加载中 三种状态
- [x] **端口信息** — 默认 6446
- [x] **模型列表** — 内置模型 + 自定义模型
- [x] **刷新按钮** — 手动刷新服务器状态和 key
- [x] **快速开始** — 内置 curl 示例和工具配置指引

### 4. ⚡ 模型测速
- [x] **一键测速** — 点击模型旁的「测速」按钮
- [x] **延迟测量** — 从请求到收到完整响应的时间 (ms)
- [x] **Token 速度** — tokens/second 生成速率
- [x] **总 Token 数** — 本次测试消耗的 tokens
- [x] **响应预览** — 显示模型返回内容的前 100 字符
- [x] **错误提示** — 限流/超时/无效模型均有明确提示
- [x] **异步不阻塞** — 多个模型可同时测速

### 5. 🔧 自定义模型
- [x] **添加模型** — 在输入框输入名称，点击「添加」
- [x] **删除模型** — 自定义模型旁有 ✕ 按钮可移除
- [x] **持久化** — 自定义列表保存到 `custom_models.json`
- [x] **自动识别** — 内置模型和自定义模型在界面上有不同标识
- [x] **测速支持** — 自定义模型同样可以测速

### 6. 🖥️ 系统托盘
- [x] **托盘图标** — 关闭窗口时最小化到托盘
- [x] **右键菜单** — Show Window / Quit
- [x] **左键点击** — 点击托盘图标恢复窗口
- [x] **后台运行** — 关闭窗口不中断代理服务器

### 7. 🔐 安全与配置
- [x] **自动建目录** — 配置文件保存在 `~/.config/opencode-free-proxy/`
- [x] **会话管理** — 每 30 分钟自动轮换 Zen API 会话 ID
- [x] **错误处理** — 上游超时/限流/空响应均有友好提示
